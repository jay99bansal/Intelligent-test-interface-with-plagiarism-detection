Detailed Description

This file of the project is applying various string matching algorithms for detecting plagiarism between subjective type questions.

In this analysis of various string matching algorithms implemented in strsim library is done and then the algorithms whose performance is good enough for the task are used combinely to come up with a fair match index between strings.
