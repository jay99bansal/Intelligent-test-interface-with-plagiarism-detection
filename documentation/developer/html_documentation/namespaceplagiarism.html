This function is taking a test id as argument and returning a list of roll numbers attempted that test.

Since test id field is not there in the table wp_postmeta, this function is extracting roll numbers by first extracting id values corresponding to that test id and whose label is roll number or similar to it. Then using these id values roll numbers can be extracted from wp_postmeta table.

This function takes plagiarism indices for individual questions and weight them according to the distribution of marks.

For each pair of roll numbers there is a plagiarism index for each question. To get one normalised Plagiarism index, these indices are weighted according to the Question's marks distribution and It finally return us a final normalised Plagiarism Indices for all pairs.

Returns
final list Contains normalised plagiarism index for each pair of roll numbers
Parameters
